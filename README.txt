Pour installer ce jeu, placez tous les fichiers labigenerator.py, jeu.py, sessions.py,
module.py et le dossier saved_game dans un même dossier,
puis lancez le fichier 'jeu.py' à l'aide de Python3.5 dans un terminal.
Le jeu a été développé et testé pour MS-DOS, il devrait fonctinner sous UNIX mais il n'y a pas été testé.
En cas de problème, vérifiez que les fichiers : labigenerator.py, jeu.py, sessions.py,
module.py et le dossier saved_game sont bien présents.
Si le problème persiste, ajoutez l'encoding et le chemin de l'interpréteur de python en tête des dits fichiers.
Pour en savoir plus, consultez les crédits (accessibles dans le menu du jeu)
